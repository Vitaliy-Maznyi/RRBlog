# RRBlog

Pet project for learning new features and technologies

# Used

rails 4.2.6

postgresql 9.5

rspec

rubocop

### Installing

git clone 
rake db:setup

## Running the tests

bundle exec rspec spec

## Check your codestyle

bundle exec rubocop

## Deployment

Deployed on heroku [http://rrmyblog.herokuapp.com/](http://rrmyblog.herokuapp.com/)
